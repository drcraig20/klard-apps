## Note for Web Platform

The web platform uses Tailwind CSS for theming, which is configured in:
- `klard-web/tailwind.config.ts` (extends theme)
- `klard-web/src/app/globals.css` (CSS variables)

This is the correct pattern for Next.js + Tailwind projects. The spec's theme constants are primarily for the React Native mobile app.
