## 1. Strategic Alignment Issues

### 1.1 Pricing Model Mismatch

**Current State:**
```
Basic: $3/mo (or free?) - Up to 5 subscriptions
Pro: $9.99/mo - Unlimited + price alerts
Saver+: $16.99/mo - BurnerCards + rules
```

**Required State (per Product Brief 2025-12-21):**
```
Pro: $12.99/mo ($119/yr annual) - Unlimited subs, email scanning, price alerts
Saver+: $19.99/mo ($179/yr annual) - + BurnerCards (4/mo), custom rules, savings reports
```

**Rationale:** Product Brief explicitly states:
> "Premium-only with 14-day free trial (no credit card required). No free tier. Research shows premium users generate 7x higher LTV ($55.21 vs $8.08)."

**Action Items:**
- [ ] Remove Basic tier entirely
- [ ] Update Pro price to $12.99/mo
- [ ] Update Saver+ price to $19.99/mo
- [ ] Add prominent "14-day free trial" messaging
- [ ] Update annual pricing to $119/yr (Pro) and $179/yr (Saver+)
- [ ] Update feature comparison table to remove Basic column

### 1.2 Trial Messaging Missing

**Current State:** "Starting at $3/month. Try Klard for 14 days."

**Required State:** "Try free for 14 days. No credit card required."

**Placement:** Should appear:
1. Hero section (primary CTA area)
2. Pricing section header
3. Each pricing card

### 1.3 Conflicting Discount Offers

**Current State:**
- Hero: "First 1,000 users get lifetime 50% off"
- Early Access section: "30% off Pro plan forever"

**Required State:** Single consistent offer across all touchpoints.

**Recommendation:** Standardize on "First 1,000 users get lifetime 50% off" (more compelling) OR align with validated strategy.

## 6. Design & UX Improvements

### 6.1 Typography Assessment

**Frontend Design Skill Guidelines:**
> "Choose fonts that are beautiful, unique, and interesting. Avoid generic fonts like Arial and Inter."

**Recommendations:**
- Verify current font stack is distinctive
- Consider display font for headlines (e.g., Cabinet Grotesk, Satoshi, General Sans)
- Body font should be refined but readable
- Ensure font pairing creates hierarchy

### 6.2 Visual Hierarchy Issues

| Issue | Current | Recommended |
|-------|---------|-------------|
| Viral moment placement | BurnerCard section (below fold) | Above fold in hero |
| "Blocked charge" notification | Small, in feature section | Large, prominent, animated |
| Trial messaging | Buried in pricing | Hero + every CTA |
| Social proof | None | Founder story or early tester quotes |

### 6.3 Motion & Micro-interactions

**Frontend Design Skill Guidelines:**
> "Focus on high-impact moments: one well-orchestrated page load with staggered reveals creates more delight than scattered micro-interactions."

**Recommendations:**
1. Hero entrance animation (staggered text reveal)
2. BurnerCard flip animation (already present - good!)
3. "Blocked charge" notification entrance (slide in from right)
4. Scroll-triggered reveals for feature sections

### 6.4 Color & Theme Consistency

Per Klard Design System:

| Token | Value | Usage |
|-------|-------|-------|
| Primary | #0D7C7A (light) / #15B5B0 (dark) | CTAs, interactive elements |
| Background | #FFFFFF (light) / #0F172A (dark) | Base surfaces |
| Success/Savings | #059669 / #10B981 | Money saved indicators |

**Verify:** All CTAs use primary teal consistently.

### 6.5 BurnerCard Visual Enhancement

The BurnerCard flip card is excellent. Enhancements:
- Larger size for mobile
- Add subtle glow/shadow for depth
- Consider showing multiple card types (One-time, Recurring, Category)
